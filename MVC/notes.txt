SEQUELIZE 
Sequelize adalah library dalam Javascript yang membuatnya mudah untuk mengelola database SQL. Sequelize adalah ORM database untuk mempermudah pemetaan database entry menjadi object, sehingga akan mempercepat dan mempermudah proses interaksi dengan database. (Object Relationship Mapping) 
Dengan memanfaatkan ORM, Anda dapat membuat aplikasi dengan mudah tanpa menuliskan perintah SQL (Structured Query Language).
